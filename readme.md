# MusicFree 桌面版的主题包

## 使用方法
1. 下载 [主题包](https://www.lanzoub.com/irK5a226qnyj)，并解压

2. 点击 + 号安装主题，安装解压文件夹内的 .mftheme 文件

3. 软件内切换主题即可
